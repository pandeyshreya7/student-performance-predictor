# 📊 Student Performance Predictor

A Machine Learning project to predict student performance based on factors like study hours, attendance, and previous scores using Python and Scikit-learn.

---

## 📌 Overview
This project analyzes student data and builds a machine learning model to predict final scores based on academic factors such as study hours, attendance, and previous performance.

---

## 🛠 Tech Stack
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

---

## ⚙️ Features
- Data loading and preprocessing
- Exploratory Data Analysis (EDA)
- Model training using Linear Regression
- Performance evaluation
- Prediction of student scores
